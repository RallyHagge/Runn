Lägg till på hemskärmen

Då öppnas sjökortet som en app, utan adressfält.

  1. Tryck på ⋯ (tre punkter) nere till höger i Safari.
  2. Välj Dela ↑.
  3. Välj Lägg till på hemskärmen ＋.
  4. Tryck Lägg till uppe till höger.
  1. Tryck på menyn ⋮ uppe till höger (Chrome).
  2. Välj Lägg till på startskärmen.
  3. Bekräfta med Lägg till.

Kom ihåg att alltid ha med ett aktuellt tryckt sjökort ombord, även om du använder det digitala.
