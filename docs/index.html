Sjökort Runn

Ange din köpkod för att öppna sjökortet.

Koden köps av Runns Båtförbund och fungerar på alla dina enheter.

Sjökort Runn

Utgivare: Runns Båtförbund (Sjökortssektionen)
Webbapp: Anders Hagman

Lägg till på hemskärmen

Då öppnas sjökortet som en app, utan adressfält.

  1. Tryck på Dela-knappen ↑ längst ner (Safari).
  2. Välj Lägg till på hemskärmen.
  3. Tryck Lägg till uppe till höger.
  1. Tryck på menyn ⋮ uppe till höger (Chrome).
  2. Välj Lägg till på startskärmen.
  3. Bekräfta med Lägg till.
